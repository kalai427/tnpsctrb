# TNPSCTRB Clone

A modern, responsive clone of TNPSCTRB built with Next.js 15+ and Vanilla CSS.

## Features
- **Modern UI/UX**: Clean layout with glassmorphism header, responsive grid, and premium typography.
- **Dynamic Routing**: Standard-wise pages (`/std-10`, `/std-12`, etc.) generated dynamically.
- **News Section**: Dedicated page for latest educational updates.
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop.
- **SEO Ready**: Semantic HTML and Metadata updates.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Vanilla CSS (CSS Modules + CSS Variables)
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

For detailed deployment instructions, please see [DEPLOYMENT.md](./DEPLOYMENT.md).

The easiest way to deploy is using [Vercel](https://vercel.com):
1. Push your code to GitHub/GitLab.
2. Import the project in Vercel.
3. Your site will be live instantly!

## Project Structure
- `app`: App Router pages and layouts.
- `components`: Reusable UI components (Navbar, Footer).
- `app/globals.css`: Global design system and variables.
